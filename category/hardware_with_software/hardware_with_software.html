The architecture is CPU type of the platform

x86_64 is 64bit base on inter x86 same to amd64
amd64 is 64bit base on inter x86 same to x86_64
x86 is 32bit base on inter x86 architecture include i386 with i686
i386 is base on x86 inter x86 architecture minimum require 386 instruction set, means before CPU is 80386 can not support, support 80386 with later CPU
i686 like i386 but minimum require 686 instruction set
armv7 is 32bit ARM v7 architecture
aarch64 is 64 bit ARM v8 architecture
ppc64le is IBM P8 little endian architecture
