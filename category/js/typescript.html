安装

每个项目独立安装:
npm install typescript --save-dev
在项目下独立使用:
npx tsc

全局安装:
npm install -g typescript
全局使用:
tsc

tsconfig.json 配置文件

在项目文件夹下放置 tsconfig.json 文件可以配置输出结果，但在指定编译文件时该配置文件不会生效
如编译 1.ts 文件
npx tsc 1.ts
此时 tsc 不会按照 tsconfig.json 配置进行输出，js文件中的变量声明都是 var
要想指定编译版本需在编译命令上加参数
npx tsc --target es2016 1.ts
此时 js 文件中的变量声明为 const 或 let
使 tsconfig.json 配置文件生效是只使用
npx tsc
