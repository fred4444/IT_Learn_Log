vim设置

全局设置在 /etc/vimrc 文件。
当前用户设置在 ~/.vimrc 文件, 若这个文件不存在则自己创建一个即可。
我的配置如下 :
set backspace=indent,eol,start
syntax on
set number
set tabstop=4
set shiftwidth=4
set cursorline
highlight cursorline cterm=none ctermbg=gray
set cursorcolumn
highlight cursorcolumn cterm=none ctermbg=gray
au BufWinLeave *.* silent mkview
au BufWinEnter *.* silent loadview


说明如下:
#避免退格键不能删除字符
set backspace=indent,eol,start
indent: 如果用了:set indent,:set ai 等自动缩进，想用退格键将字段缩进的删掉，必须设置这个选项。否则不响应。
eol:如果插入模式下在行开头，想通过退格键合并两行，需要设置eol。
start：要想删除此次插入前的输入，需设置这个。
#语法高亮
syntax on
#显示行号
set number
#or
set nu
#设置制表位等效的空格数, 即按下 tab 键空几个格
set tabstop=4
#or
set ts=4
#设置自动缩进所使用的空白长度
set shiftwidth=4
#or
set sw=4
#显示光标行高亮
set cursorline
#设置光标行高亮颜色
highlight cursorline cterm=none ctermbg=gray
#设置光标高亮的详细解释如下:
hi CursorLine cterm=NONE ctermbg=darkred ctermfg=white guibg=darkred guifg=white
其中Cursorline和CursorColumn分别表示光标所在的行和列
----------------------------------------------------------------------------------
Vim识别三种不同的终端：term，黑白终端；cterm，彩色终端；gui，Gvim窗口。
term，可以定义其字体显示为：bold、underline、reverse、italic或standout。例如以下命令，用逗号来组合使用这些属性：
:highlight Keyword term=reverse,bold
cterm，可以用ctermfg设置前景色；用ctermbg设置背景色。可以使用help highlight 来查看相关信息
例如以下命令，定义蓝底红字并使用下划线来显示注释：
:highlight Comment cterm=underline ctermfg=red ctermbg=blue
例如以下命令，定义选中的行前景背景色反转：
:highlight cursorline cterm=reverse ctermfg=none ctermbg=none
或:
:highlight cursorline cterm=reverse
gui，可以使用选项gui=attribute，来定义图形窗口下语法元素的显示属性。选项guifg和guibg，用来定义了前景色和背景色。推荐使 用的颜色包括：black, brown, grey, blue, green, cyan, magenta, yellow, white。

正则表达式中使用变量

用\(和\)包起来的部分所匹配的值就是变量的值，在替换的新的值中用\1，来表示第一个变量，\2表示第2个变量，如此类推.

16进制格式

按ESC进入命令模式按:在左下角使用xxd命令把文件转换成十六进制格式
%!xxd

按ESC进入命令模式按:在左下角使用xxd命令把文件从十六进制格式转换成文本格式
%!xxd -r

用命名缓冲区复制粘贴

在vim中有26个命名缓冲区名字为a~z(注意这26个命名缓存区与其它操作的命名缓冲区是共享的如在a缓冲区中录了一个宏，然后又将复制的值放到了a缓冲区中则原先的宏的内容被覆盖)在删除和复制时可以指定缓冲区将值暂存在指定的缓冲区中，在粘贴时在通过指定缓冲区名称来粘贴指定缓冲区中的内容。

复制到命名缓冲区
举例，将当前光标所在位置及其后共3行内容复制到命名缓冲区a
按ESC进入命令模式，按"a3yy

在命令行模式中用
起始行号, 末尾行号 yank 缓冲区名称
将指定行范围的内容复制到指定名称的缓冲区中
举例，将10到20的内容复制到命名缓冲区a中
:10, 20 yank a
若不指定缓冲区名称则复制到默认缓冲区中不在这26个命名缓冲区中
举例，复制10到20的内容
:10, 20 yank

从命名缓冲区中粘贴
举例，将命名缓冲区a中的内容粘贴到当前光标所在位置
按ESC进入命令模式，按"ap

录制和重放宏

在vim中有26个命名缓冲区名字为a~z(注意这26个命名缓存区与其它操作的命名缓冲区是共享的如在a缓冲区中录了一个宏，然后又将复制的值放到了a缓冲区中则原先的宏的内容被覆盖)宏可以录制一组操作，然后重放宏，在自动重复做这种操作一次或多次。

录制宏
举例，将宏录制到命名缓冲区a中。
按ESC进入命令模式，按q再按a然后在左下角会出现recording表明处于录制状态，在这之后我们做的操作将会被录到命名缓冲区a中，操作做完后按ESC进入命令模式，按q结束录制。

重放宏
举例，重放命名缓冲区a中的宏。
按ESC进入命令模式，按@再按a。
