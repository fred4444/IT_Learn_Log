termux

Termux是一个Android下一个高级的终端模拟器,开源且不需要root,支持apt管理软件包.
完美支持Python,PHP,Ruby,Go,Nodejs,MySQL等

软件管理

搜索包
pkg search <query>

安装包
pkg install <package>

卸载包
pkg uninstall <package>

重新安装包
pkg reinstall <package>

更新源
pkg update

升级软件包
pkg upgrade

列出可供安装的所有包
pkg list-all

列出已经安装的包
pkg list-installed

显示某个包的详细信息
pkg show <package>

显示某个包的相关文件夹路径
pkg files <package>

安装软件vim, git, nodejs, clang

pkg install vim

pkg install git

pkg install nodejs

pkg install clang

访问外存, access sdcard, like usb disk

termux-setup-storage
或是先
install termux-tools package
pkg install termux-tools

生成密钥对

install openssh package
pkg install openssh
安装后会自动生成各种类型的蜜钥, 蜜钥文件的路径在
/data/data/com.termux/fils/usr/etc/ssh/
下, 将rsa类型的密钥文件复制到~/.ssh/路径下
cp ~/../usr/etc/ssh/ssh_host_rsa_key.pub ~/.ssh/id_rsa.pub
cp ~/../usr/etc/ssh/ssh_host_rsa_key ~/.ssh/id_rsa
