install printer

install network printer to local machine

step one:

open web broswer

for example:

firefox &

step two:

open cups web page, the url is:

http://localhost:631

step three:

add printer, click the link "adding Printers and Classes", and then jump to "Administration" tag, click the "Add Printer" button, chose the "AppSocket/HP JetDirect" option, input socket://printer_server_ip_address at the "Connection", input the Name, Description, Location, three argument, then click the continue button, select the compatible driver, click "Add Printer" button.

install network printer to server machine(minimal install mode, only commond line mode)

step one:

use ssh connect to server machine

step two:

change cups config file change the listen ip address, the file path is "/etc/cups/cupsd.conf", change "Listen localhost:631" to "Listen *:631"

step three:

change cups config file addition the access right, the file path is "/etc/cups/cupsd.conf", change:

<Location />
# Allow remote administration...
Order allow,deny
Allow @LOCAL
</Location>
<Location /admin>
# Allow remote administration...
Order allow,deny
Allow @LOCAL
</Location>
<Location /admin/conf>
AuthType Default
Require user @SYSTEM
# Allow remote access to the configuration files...
Order allow,deny
Allow @LOCAL
</Location>


to:


<Location />
# Allow remote administration...
Order allow,deny
Allow all
</Location>
<Location /admin>
# Allow remote administration...
Order allow,deny
Allow all
</Location>
<Location /admin/conf>
AuthType Default
Require user @SYSTEM
# Allow remote access to the configuration files...
Order allow,deny
Allow all
</Location>

if not do this step will got "403 forbidden" error

step four:

change cups config file disable the ssl, the file path is "/etc/cups/cupsd.conf", addition "DefaultEncryption IfRequested" to file at anyline

if not do this step will got "CUPS: 426 Upgrade Required" error

step five:

addition printer same to the local addition printer, but change the localhost to server machine ip address
